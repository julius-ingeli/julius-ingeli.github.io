Vulnerability Store

This project served as an artifact for my Bachelor's thesis, which focused on the development of a practical solution for the management and storage of vulnerabilities. The project aimed to create a centralized repository that allows for efficient tracking, categorization, and retrieval of vulnerability information. The system was designed to support security professionals in identifying and mitigating potential threats by providing a structured and accessible platform for vulnerability data. It was a baseline for a DevSecOps solution for companies to manage their vulnerabilities and security issues in a more efficient way. The project was developed using modern web technologies and follows best practices in software development, ensuring scalability, security, and usability.
Uses: Python, Flask, Uvicorn, Docker, PostreSQL
